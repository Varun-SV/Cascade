Multi-tier AI agent orchestration

Cascade coordinates planner, worker, and local models for coding tasks.

This project is a Python library and CLI that delegates work across model tiers to balance capability, speed, and cost. It includes provider integrations, tool execution, escalation, and optional budget tracking.

Get Started Read README

Architecture

T1 Planner High-capability orchestration and task planning
T2 Worker Execution-focused model for delegated subtasks
T3 Local Executor Fast local model via Ollama for lightweight actions

Project Overview

Cascade is organized as a Python package with modules for core agent orchestration, provider integrations, tool registries, configuration loading, budget tracking, and a Typer-based CLI. The repository also includes tests for configuration, escalation, and tool behavior.

Core package

cascade/ contains the public API, CLI, providers, tools, orchestration logic, and utilities.

Testing

tests/ covers tool execution, config loading, and escalation-related behavior.

Features

Based on the repository contents and README, these are the main capabilities of the project.

Three-tier delegation

Planner, worker, and local executor roles allow tasks to cascade to the right capability level.

Multiple providers

Supports Anthropic, OpenAI, Google, and Ollama through dedicated provider modules.

Built-in tools

Includes file operations, shell commands, code search, git actions, and web access.

Escalation policy

Confidence-based escalation helps agents hand work upward when needed.

Budget tracking

Optional session and per-model cost controls are available through the budget tracker.

CLI + Python API

Use Cascade from the command line or integrate it directly into Python applications.

Quick Start

The project is packaged with pyproject.toml and exposes a cascade CLI entry point.

pip install -e .
cascade init
cascade run "list all Python files and count lines of code"

Python API

The public entry point is the Cascade class exported from cascade.__init__.

from cascade import Cascade

agent = Cascade(config_path="./cascade.yaml")
result = agent.run("add error handling to auth.py")

print(result.summary)
print(f"Cost: ${result.total_cost:.4f}")

CLI Commands

Run a task

cascade run "task"

List models

cascade models

Show config

cascade config-info

Initialize config

cascade init

cascade run "task" --budget 0.50
cascade run "task" --verbose
cascade run "task" --root ./myproject
